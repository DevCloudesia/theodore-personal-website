# Planet STL Models

Place your STL files in this directory with the following naming convention:

- `Dangerball 3d design (1).stl` - Dangerball planet model
- `Paradise 3d design (1).stl` - Paradise planet model
- `Coldi 3d design (1).stl` - Coldi planet model
- `Gassyball 3d design (1).stl` - Gassyball planet model
- `Gag 3d design (1).stl` - Gag planet model
- `Neptwo 3d design (1).stl` - Neptwo planet model

The 3D System viewer will automatically load these models on startup.
